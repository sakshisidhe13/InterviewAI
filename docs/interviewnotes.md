# Interview Notes

## Git

### What is Git?

Git is a version control system that tracks changes made to a project over time.

### What is GitHub?

GitHub is a cloud platform that stores Git repositories online.

### What is a Repository?

A repository is a project managed by Git.

### What is git init?

It initializes a Git repository in a folder and creates a hidden `.git` directory to track project history.

### What is git remote add origin?

It connects the local Git repository on your computer to a remote repository on GitHub.
